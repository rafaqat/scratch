Expose database operations via MCP so AI agents can work with structured data.

## Acceptance Criteria

- [ ] `db_list` — List all databases with schema summaries
- [ ] `db_get_schema` — Get full schema for a database
- [ ] `db_query` — Query rows with filters, sorting, pagination
- [ ] `db_insert_row` — Create a new row with field values
- [ ] `db_update_row` — Update specific fields on a row (etag for concurrency)
- [ ] `db_delete_row` — Delete a row
- [ ] `db_create` — Create a new database with schema definition

## Technical Notes

Follow the existing MCP tool pattern in `src-tauri/src/mcp.rs`. Use the same JSON-RPC dispatch pattern as the stories tools.
