Note templates stored as markdown files with variable substitution.

## Acceptance Criteria

- [ ] Templates stored in `.scratch/templates/*.md`
- [ ] "New from template" option in command palette (`Cmd+P`)
- [ ] "New from template" in new-note dropdown/context menu
- [ ] Variable substitution: `{{date}}`, `{{date:YYYY-MM-DD}}`, `{{title}}`, `{{time}}`
- [ ] `{{cursor}}` marks initial cursor position after creation
- [ ] Ship 3 built-in templates: Daily Journal, Meeting Notes, Project Brief
- [ ] Users can create/edit templates (they're just markdown files)

## Technical Notes

Templates are plain markdown files — no special runtime needed. On "new from template": read template, substitute variables, create note, place cursor. Command palette already supports custom commands — add template list as a command group.
