URL preview cards with title, description, favicon.

## Acceptance Criteria

- [ ] `/bookmark` slash command or paste URL to create bookmark block
- [ ] Fetches page metadata (title, description, image, favicon) via Tauri backend
- [ ] Card display: favicon + title + description + domain
- [ ] Click opens URL in default browser
- [ ] Graceful fallback when metadata fetch fails (show URL only)
- [ ] Cached metadata in frontmatter comment to avoid re-fetching

## Technical Notes

Tauri backend command to fetch URL metadata (use `reqwest` + HTML parsing for og:tags). Cache result in a markdown comment block above the bookmark reference. BlockNote custom block with `createReactBlockSpec`.
