Custom BlockNote block that renders database rows as an editable table.

## Acceptance Criteria

* [ ] `/database` slash command inserts a table view block
* [ ] Renders rows from a linked database folder as editable table
* [ ] Column sorting (click header)
* [ ] Column filtering (dropdown per column)
* [ ] Inline cell editing with type-aware inputs
* [ ] Add/delete rows from table UI
* [ ] Add/remove/reorder columns
* [ ] Serializes to markdown as a reference: `[database:my-tasks](view:table)`

## Technical Notes

Use BlockNote `createReactBlockSpec` for the custom block. Fetch data via Tauri commands. Table rendering with Tailwind — no heavy table library needed.
