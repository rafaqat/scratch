Pre-defined row templates for databases.

## Acceptance Criteria

* [x] Database schema supports a `templates` section with named row templates
* [x] "New row from template" option in table/board view
* [x] Template pre-fills frontmatter fields with default values
* [x] Template can include markdown body content
* [x] Example: "Bug Report" template with status=Backlog, tags=[bug], body with repro steps

## Technical Notes

Store templates in the database `_schema.md` frontmatter:

```yaml
templates:
  bug-report:
    title: "Bug: {{title}}"
    fields: {status: Backlog, tags: [bug]}
    body: |
      ## Steps to Reproduce
      1.
      ## Expected
      ## Actual
```
