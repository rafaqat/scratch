Pre-defined row templates for databases.

## Acceptance Criteria

- [ ] Database schema supports a `templates` section with named row templates
- [ ] "New row from template" option in table/board view
- [ ] Template pre-fills frontmatter fields with default values
- [ ] Template can include markdown body content
- [ ] Example: "Bug Report" template with status=Backlog, tags=[bug], body with repro steps

## Technical Notes

Store templates in the database `_schema.md` frontmatter:
```yaml
templates:
  bug-report:
    title: "Bug: {{title}}"
    fields: {status: Backlog, tags: [bug]}
    body: |
      ## Steps to Reproduce
      1.
      ## Expected
      ## Actual
```
